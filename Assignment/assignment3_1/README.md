# 분수 추상 자료형(ADT)

## 개요

- 정수형 분자(numerator)와 분모(denominator)로 이루어진 분수를 추상화한 자료형이다.
- 모든 연산의 결과는 기약분수 형태로 반환된다.
- 분모가 0인 경우는 허용되지 않는다.

## 자료형 구조

- int numerator // 분자 -> 정수
- int denominator // 분모 -> 0되면 안됨

## 연산 목록

### 1. fromString - 문자열을 숫자로 변환

하는 일: "2/3" 문자열을 읽어서 숫자로 바꿔줌.
이유; 사용자가 입력한 값은 문자열 형태이기 때문에, 컴퓨터가 계산할 수 았게 분수 형태로 바꿔주는 과정이 필요하다.
입력: 처음 입력시 "A/B" 같은 문자열로 들어옴.
출력: Fraction
예시: 2/3 입력 -> 분자 2, 분모 3인 분수.

### 2.toString - 분수를 다시 문자열로 변환

하는 일: 내부에 저장된 분수를 문자열로 다시 변환시킴.
이유: 프로그램 내부에서 분수는 숫자 두개로 저장되어 있어서 그냥은 화면에 출력할 수 없어서 사람이 읽을 수 있는 형태로 변환하는 과정이 필요하다.
입력: 분수 하나
결과: "A/B" 형태의 문자열을 돌려준다.
예시: 분자 3, 분모 4인 분수 -> "2/3"

### 3.reduce - 기약분수로 줄이기

하는 일: 분수를 기약분수로 만든다.
결과: 약분된 분수를 돌려준다.
예시: 4/6 -> 2/3 (분자,분모 최대공약수 2로 나눔)

#### 4.add - 덧셈

하는 일: 두 분수를 더한다.
입력: 분수 두개 (f1,f2)
결과: f1 + f2의 결과를 기약분수로 돌려준다.
예시: 1/2 + 1/3 -> 5/6

### 5.subtract - 뺄셈

하는 일: 두 분수를 뺀다.
입력: 분수 두개 (f1,f2)
결과: f1 - f2의 결과를 기약분수로 돌려준다.
예시: 3/4 - 1/4 → 1/2

### 6.multiply - 곱셈

하는 일: 두 분수를 곱함.
입력: 분수 두개 (f1,f2)
결과: f1 _ f2의 결과를 기약분수로 돌려준다.
예시: 1/2 _ 1/3 → 1/6

### 7.divide - 나눗셈

하는 일: 두 분수로 나눈다.
입력: 분수 두개 (f1,f2)
결과: f1 / f2의 결과를 기약분수로 돌려준다.
예시: 1/2 / 1/4 → 2/1
