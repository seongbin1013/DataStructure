# GCD(최대공약수), LCM(최소공배수)

## GCD(최대공약수)

정의: 두 정수를 나누어 떨어지게 하는 가장 큰 정수이다.

예: GCD(12,8) = 4

### 알고리즘: 유클리드 호제

-> 두 수 a,b가 있을 때, a를 b로 나눈 나머지가 0이 될 때까지 반복함.
나머지가 0이 되는 수간의 b가 최대공약수이다.

```
GCD(a,b):
입력: 정수 a,b (b>0)
출력: a와 b의 최대공약수

반복:
r <- a % b (a를 b로 나눈 나머지)
if r == 0;
return b -> 나머지가 0이면 b가 최대공약수
a <- b
b <- r
```

### 예시: gcd(48,18)

```
48 % 18 = 12 -> a=18, b=12
18 % 12 = 6 -> a=12, b=6
12 % 6 = 0 -> 나머지 0, 정답 = 6
```

### LCM(최소공배수)

정의: 두 정수의 공통 배수 중 가장 작은 정수이다.

알고리즘: 최소공배수는 GCD를 이용하여 바로 구할 수 있음.

```
LCM(a,b) = a x b / GCD(a,b)
```

이유: 두 수의 곱 a x b에는 공약수 부분이 두 번 포함되어 있기 때문에, GCD로 한번 나눠주면 겹치는 부분을 제거 할 수 있음.

```
LCM(a,b):
입력: 정수 a,b
출력: a와 b의 최소공배수

g <- GCD(a,b)
return(a/g) x b
```

### 예시: LCM(4,6)

```
GCD(4,6) = 2
LCM = (4/2) x 6 = 2 x 6 = 12
```
