# GCD(최대공약수), LCM(최소공배수)

## GCD(최대공약수)

정의: 두 정수를 나누어 떨어지게 하는 가장 큰 정수이다.

예: GCD(12,8) = 4

### 알고리즘: 유클리드 호제

-> 두 수 a,b가 있을 때, a를 b로 나눈 나머지가 0이 될 때까지 반복함.
나머지가 0이 되는 수간의 b가 최대공약수이다.

```
GCD(a,b):
입력: 정수 a,b (b>0)
출력: a와 b의 최대공약수

반복:
r <- a % b (a를 b로 나눈 나머지)
if r == 0;
return b -> 나머지가 0이면 b가 최대공약수
a <- b
b <- r
```

### 예시: gcd(48,18)

```
48 % 18 = 12 -> a=18, b=12
18 % 12 = 6 -> a=12, b=6
12 % 6 = 0 -> 나머지 0, 정답 = 6
```
