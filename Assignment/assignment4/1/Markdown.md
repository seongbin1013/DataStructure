# 과제 4-1: GCD(최대공약수) 시간복잡도와 Big-O

## 1. 제출한 GCD 알고리즘

- **파일 링크**: [my_math.c](./my_math.c) / [my_math.h](./my_math.h)
- **알고리즘 방식**: 유클리드 호제법 (반복문)

```c
int gcd(int a, int b) {
    if(a < 0) a = -a;
    if(b < 0) b = -b;

    while(b != 0) {      // ← 핵심 루프
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```

---

## 2. 알고리즘 동작 원리

유클리드 호제법은 다음 성질을 반복 적용한다.

```
GCD(a,b) = GCD(b, a mod b) (단, b != 0)
GCD(a,0) = a
```

매 반복마다 b가 a mod b로 교체되므로, 나머지가 0이 될 떄가지 반복하다.

### 예시: GCD(21,13)

| 단계 |  a  |  b  | a mod b  |
| :--: | :-: | :-: | :------: |
|  1   | 21  | 13  |    8     |
|  2   | 13  |  8  |    5     |
|  3   |  8  |  5  |    3     |
|  4   |  5  |  3  |    2     |
|  5   |  3  |  2  |    1     |
|  6   |  2  |  1  | 0 → 종료 |
